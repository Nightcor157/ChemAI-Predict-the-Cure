# ChemAI: Predict the Cure

Проект для хакатона **ChemAI: Predict the Cure**. Цель — построить воспроизводимый ML-пайплайн для предсказания трёх показателей биологической активности химических соединений против вируса гриппа:

- `IC50` — концентрация, при которой вещество подавляет 50% активности вируса;
- `CC50` — концентрация, при которой вещество токсично для 50% клеток;
- `SI` — индекс селективности.

Метрика соревнования — RMSE. Текущая отправленная версия решения даёт Kaggle score около **301.11499**. Более агрессивные эксперименты с подстановкой повторяющихся молекул и расчётом `SI = CC50 / IC50` ухудшили результат, поэтому в финальной версии оставлен стабильный и воспроизводимый RandomForest-подход.

## Подход

Решение сделано в простом и воспроизводимом стиле:

1. данные читаются напрямую из Kaggle input;
2. удаляется служебный столбец `index`;
3. пропуски в признаках заполняются медианой;
4. для каждого таргета обучается отдельная модель;
5. итоговые предсказания собираются в формате `sample_submission`.

Используется `RandomForestRegressor`, потому что он устойчив на табличных дескрипторах, не требует сложного масштабирования признаков и хорошо работает на небольших выборках.

## Модели по таргетам

| Таргет | Обработка целевой переменной | Модель |
|---|---|---|
| `IC50` | `PowerTransformer(method='box-cox')` | `RandomForestRegressor(n_estimators=300, min_samples_leaf=3)` |
| `CC50` | `RobustScaler()` | `RandomForestRegressor(n_estimators=300, min_samples_leaf=3)` |
| `SI` | фильтрация 1% крайних значений + `PowerTransformer(method='box-cox')` | `RandomForestRegressor(n_estimators=100, max_depth=10)` |

Такой подход выбран из-за различий в распределениях таргетов: `IC50` и `SI` имеют выраженную асимметрию, а `CC50` содержит крупные значения и чувствителен к выбросам.
